# 🤖 AI Dev Agent

Cria aplicações completas do zero e implementa features automaticamente usando IA.

## ✨ Features

- ✅ **CREATE**: Cria APIs, webapps e scripts do zero
- ✅ **IMPLEMENT**: Implementa features em projetos existentes
- ✅ **TESTES**: Gera testes unitários automaticamente
- ✅ **CUSTO ZERO**: Usa Gemini (grátis) + Ollama (local)

## 🚀 Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/ai-dev-agent.git
cd ai-dev-agent

# Instale dependências
pip install -r requirements.txt

# Configure .env
cp .env.example .env
# Edite .env com suas API keys
```
